class == 자바의 기본단위

class 단위로 예제를 작성할 예정

class라는 예제들을 묶어서 정리할 필요 >> package 패키지에 저장

동작시킬 main이 필요함

package는 class들을 보관하는 단위
public : 공개범위를 의미함. <-> private
	모두에게 공개 <-> 모두에게 은닉
	java를 처음 개발한 사람들은 "모든 개발자들에게 널리 이롭게 할 코드"가 필요해서 만듬
	java는 기본적으로 공개 정책(public) <-> C는 기본적으로 정보은닉체제(private)

클래스명 규칙 : 대문자로 시작 <-> 메소드, 변수,... 소문자로 시작

함수 : input, output, 기능을 가진 것
	▶ output 함수명(input)
	함수 이름은 반드시 기능을 유추할 수 있도록 지어야한다.
	함수는 input을 담아야하기 때문에 반드시 ( )가 있어야 한다.

input : 인자, 인수, 입력값, 매개변수, arguments==args, 파라미터, parameter
output : 출력값, 결과값, return 반환값

특정 단어, 키워드에 대한 별칭을 잘 알아둘 필요가 있다.

변수
